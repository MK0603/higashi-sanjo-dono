# 東三条殿 (Higashi-sanjo-dono)

このプロジェクトは、平安時代の藤原氏の邸宅である「東三条殿」を3Dボクセルジオラマで再現したものです。
寝殿（本殿）から庭園を眺めた際の、奥行きのある情緒豊かな景色を表現しています。

## 概要

*   **テーマ**: 藤原家の邸宅「東三条殿」の庭園風景
*   **視点**: 寝殿から南側（庭園・池）を眺めた景観
*   **技術**: Three.js / TypeScript / Vite
